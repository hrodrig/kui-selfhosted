# Contributing

See the application repos for code changes:

- **[kui](https://github.com/hrodrig/kui)** — dashboard UI, auth, Dockerfile
- **[kiko](https://github.com/hrodrig/kiko)** — collector, stats API

This repository accepts deployment manifest improvements (Compose, Helm, Kubernetes YAML, docs).

Use **English** for commits and documentation. Work on **`develop`**; release snapshots on **`main`** with tag **`v<VERSION>`**.
