# Security

Report deployment-related security issues for this repository via [GitHub Security Advisories](https://github.com/hrodrig/kui-selfhosted/security/advisories) or private contact with the maintainer.

**Do not commit real secrets.** Use `${STACK_HOST_DATA}/.env` or Kubernetes Secrets. Rotate **`KIKO_API_KEY`** and **`KUI_ADMIN_PASSWORD`** in production.

Application vulnerabilities belong in **[kui](https://github.com/hrodrig/kui)** or **[kiko](https://github.com/hrodrig/kiko)**.
